Menu component

The left side menu of this page is rendered using the png-menu

Usage

To render the menu the code is quite simple 

        
  <png-menu
    [menuTree]="menuTree"
  ></png-menu>

The component.ts that will make use of this menu must pass to it a MenuTree structure. The following is an example 

        
{{typescriptComponent}}

Please note that that MenuTree class is imported from ./models/menu and it has the following definition
